# Learning Platform - Backend API

FastAPI backend for an AI-powered learning platform with roadmap-style course progression.

## Features

- 🔐 JWT Authentication (Student & Teacher roles)
- 📚 Course Management with Roadmap UI support
- 💳 Payment Integration (Chapa for Ethiopia, Stripe for International)
- 💬 Real-time Chat with WebSocket
- 🤖 AI-Powered Recommendations
- 📊 Progress Tracking

## Tech Stack

- **Framework**: FastAPI
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy
- **Authentication**: JWT (python-jose)
- **Payments**: Chapa & Stripe
- **Real-time**: WebSockets
