##  Overview

This is the frontend for an AI-powered learning platform that provides:
- **Personalized learning paths** generated by AI
- **Interactive course roadmaps** with progress tracking
- **Real-time chat** for student collaboration
- **AI assistant** for learning support
- **Secure payments** with Chapa (Ethiopia) and Stripe (International)
- **Role-based dashboards** for students and teachers

## Features

### For Students
- AI-generated personalized learning roadmaps
- Browse and enroll in courses
- Track learning progress with visual indicators
- Real-time chat with peers and instructors
-  AI learning assistant for questions
- Responsive design for mobile learning
-  Secure payment gateway

### For Teachers
-  Create and manage courses
- Add lessons with structured modules
- Track student enrollment and progress
- Communicate with students via chat
- View course analytics

### General Features
-  JWT authentication
- Modern, responsive UI
- Interactive course roadmaps
- Real-time notifications
- File sharing in chat
-  Multi-currency payment support

## Tech Stack

| Category | Technology |
|----------|------------|
| Framework | React 18 |
| Routing | React Router DOM v6 |
| State Management | React Context API |
| Styling | CSS Modules / Custom CSS |
| Icons | React Icons (Fa, Io, Md, Bs, Rx, Ai) |
| HTTP Client | Axios |
| Real-time | WebSocket API |
| Payments | Chapa (ETB), Stripe (USD) |
| Authentication | JWT Bearer Token |
